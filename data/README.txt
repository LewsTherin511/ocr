Video file top-100-shots-rallies-2018-atp-season.mp4 contains 100 snippets of different (but sometimes similar)
tennis matches.

Inside top-100-shots-rallies-2018-atp-season-scoreboard-annotations.json are annotations of the scoreboard saved within
a dictionary with frame numbers as keys and annotation data as values. Keys have to be of type string because of JSON
parsing constraints. If some frame number is not present inside the dictionary that means that there is no scoreboard
present in that frame.
